# magnaprobe-pytools

Lightweight Python tools for parsing, quality-controlling, and visualizing [MagnaProbe](https://agupubs.onlinelibrary.wiley.com/doi/full/10.1029/2018WR023559) snow depth data.

## Overview

`magnaprobe-pytools` is a small collection of Python utilities for working with Magnaprobe snow depth measurements exported.

## Features

The tools will support:

- Exploratory data analysis (EDA): Generate histograms, summary statistics, and depth distributions
