# magnaprobe-pytools

Lightweight Python tools for parsing, quality-controlling, and visualizing [https://agupubs.onlinelibrary.wiley.com/doi/full/10.1029/2018WR023559](Magnaprobe) snow depth data

## Overview

`magnaprobe-pytools` is a small collection of Python utilities for working with Magnaprobe snow depth measurements exported from Campbell Scientific data loggers (TOA5 format).
